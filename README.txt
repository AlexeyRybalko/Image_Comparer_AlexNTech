Image Comparer

Что это?
- Десктопное приложение для сравнения двух изображений.
- Поддерживает режимы slide и side-by-side.
- Позволяет загружать изображения разными способами, масштабировать их, перемещать, экспортировать текущий вид и быстро возвращаться к недавним файлам.

Запуск:
- Откройте ImageComparer.exe
- Не удаляйте и не перемещайте папку assets отдельно от приложения.

Основные возможности:
- Сравнение двух изображений с вертикальным разделителем в режиме slide.
- Режим side-by-side: левое и правое изображение показываются в отдельных половинах окна.
- Экспорт PNG текущей видимой области сравнения.
- Мини карта при большом увеличении.
- История недавних изображений в нижней карусели.
- Светлая и тёмная тема.
- Скрытие интерфейса для чистого просмотра.

Режимы сравнения:
- Slide
  Оба изображения накладываются друг на друга. Перемещение вертикального разделителя показывает разницу между изображениями. 
- Side-by-side
  Одно изображение размещается слева, второе справа. Каждую сторону можно масштабировать отдельно.

Если пропорции сильно различаются.
- До определенноЙ разницы в пропорциях сторон изображений они будут деформироваться для лучшего наложения и проведения сравнения.
- Если разница пропорций изображений слишком большая, автоматически включается natural aspect ratio mode.
- В этом режиме изображения накладываются без деформации, с сохранением своих оригинальных пропорций.
- Дополнительно появляется верхняя временная плашка с уведомлением.

Варианты Загрузки изображений:
- Кнопки Load A и Load B.
- Клик по стартовым зонам A и B при пустом состоянии.
- Drag-and-drop файлов из проводника в левую или правую половину окна.
- Drag-and-drop из нижней карусели Recents.
- Вставка из буфера через Ctrl+V.
  Изображение вставляется в левую или правую половину в зависимости от положения курсора.

Управление
- Левый клик в режиме slide:
  ставит разделитель под курсор и сразу позволяет тянуть его.
- Правая кнопка мыши в режиме slide:
  перемещает текущий вид.
- Колесо мыши в режиме slide:
  приближает и отдаляет.
- В side-by-side колесо над каждой половиной масштабирует только эту сторону.
- Правая кнопка мыши в side-by-side:
  двигает левое или правое изображение отдельно, в зависимости от стороны курсора.
- Reset View:
  сбрасывает масштаб и смещение.

Мини карта
- Появляется только в режиме slide.
- Появляется при увеличении больше 300%.
- Скрывается при уменьшении до 200% и ниже.
- Красная рамка показывает текущую видимую область.
- По мини карте можно перемещаться зажатой левой кнопкой мыши без изменения масштаба.

Список недавно использованных изображений:
- По умолчанию скрыт.
- Показывается кнопкой Show Recents.
- Содержит недавние изображения без дублей по имени.
- Прокрутка:
  стрелками
  нижним слайдером
  колесом мыши при наведении на карусель
- Из карусели можно перетаскивать изображения в рабочую область (левую или правую).

Экспорт
- Кнопка Export PNG активна только в режиме slide и только когда загружены оба изображения.
- Сохраняет PNG только из области сравнения: изображения и разделитель, без остального UI.

Темы
- Кнопка Day theme / Night theme переключает светлую и тёмную тему.

Скрытие интерфейса
- Кнопка Hide UI скрывает весь интерфейс, оставляя только область изображений.
- Кнопка Show UI возвращает интерфейс обратно.
- Горячая клавиша: F11

Горячие клавиши
- Ctrl+V
  Вставить изображение из буфера.
- F11
  Скрыть или вернуть интерфейс.

Нижняя информационная строка
- В обычном режиме показывает статус сравнения и масштаб.
- В side-by-side рядом с разрешением каждого изображения показывается текущий масштаб этой стороны в процентах.


English

What it is
- A desktop app for comparing two images.
- Supports slide and side-by-side modes.
- Lets you load images in several ways, zoom, pan, export the current view, and quickly reuse recent files.

Launch
- Open ImageComparer.exe
- Do not remove or move the assets folder separately from the application.

Core features
- Two-image comparison with a vertical divider in slide mode.
- Side-by-side mode: left and right images are shown in separate halves of the window.
- PNG export of the current visible comparison area.
- Mini-map at high zoom levels.
- Recent images carousel at the bottom.
- Light and dark themes.
- Hide UI for a clean viewing mode.

Comparison modes
- Slide
  Both images are layered on top of each other, and the vertical divider defines the boundary between them.
- Side-by-side
  One image is shown on the left, the other on the right. Each side can be zoomed independently.

When aspect ratios differ a lot
- If the aspect ratio difference is too large, the app automatically switches to natural aspect mode.
- In this mode the images are layered without deformation, keeping their original proportions.
- A temporary top notification pill also appears.

Loading images
- Load A and Load B buttons.
- Click the A and B start zones when the app is empty.
- Drag and drop files from Explorer into the left or right half of the window.
- Drag and drop from the Recents carousel.
- Paste from clipboard with Ctrl+V.
  The image is inserted into the left or right side based on the cursor position.

Controls
- Left click in slide mode:
  places the divider under the cursor and lets you drag it immediately.
- Right mouse button in slide mode:
  pans the current view.
- Mouse wheel in slide mode:
  zooms in and out.
- In side-by-side mode:
  the mouse wheel over each half zooms only that side.
- Right mouse button in side-by-side:
  pans the left or right image independently depending on the cursor side.
- Reset View:
  resets zoom and panning.

Mini-map
- Appears only in slide mode.
- Appears above 300% zoom.
- Hides again at 200% zoom or below.
- A red rectangle shows the current visible area.
- You can drag inside the mini-map with the left mouse button to move the view without changing zoom.

Recents
- Hidden by default.
- Open with Show Recents.
- Stores recent images without duplicate file names.
- Scrolling methods:
  arrows
  bottom scrubber slider
  mouse wheel when the cursor is over the carousel
- You can drag images from the carousel into the working area.

Export
- Export PNG is active only in slide mode and only when both images are loaded.
- It saves only the comparison area: images and divider, without the rest of the UI.

Themes
- The Day theme / Night theme button switches between light and dark themes.

Hide UI
- Hide UI hides the full interface and leaves only the image area.
- Show UI brings the interface back.
- Hotkey: F11

Hotkeys
- Ctrl+V
  Paste an image from the clipboard.
- F11
  Hide or restore the interface.

Bottom info line
- In normal mode it shows comparison status and zoom.
- In side-by-side mode each image also shows its own zoom percentage next to its resolution.
